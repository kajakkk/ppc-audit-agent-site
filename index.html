PPC Audit Agent

PPC Audit Agent is a software platform that analyzes advertising accounts and generates automated audit reports.

Google Ads API Usage

The application uses the Google Ads API in read-only mode to retrieve campaign, ad group, keyword, conversion and performance data.

Retrieved data is used to generate audit reports and optimization recommendations.

The application does not automatically create, edit or delete advertising entities.
