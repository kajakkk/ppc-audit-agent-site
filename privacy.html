Privacy Policy

Last updated: June 12, 2026

PPC Audit Agent is a software tool developed by Karel Kašpárek. This Privacy Policy explains how the application handles data.

Data We Access

PPC Audit Agent may access advertising account data through official advertising platform APIs, including the Google Ads API.

This may include campaign data, ad group data, keyword data, asset data, conversion data and performance metrics.

How We Use Data

The data is used only to generate advertising account audit reports and optimization recommendations for the authorized user.

Data Sharing

We do not sell user data. We do not share user data with third parties except where required to provide the service or comply with legal obligations.

Google Ads API

PPC Audit Agent uses the Google Ads API in read-only mode. The application does not automatically create, edit or delete campaigns, ads, keywords, budgets or other advertising entities.

Data Retention

Data is retained only for as long as necessary to provide audit functionality, troubleshoot issues, or comply with legal requirements.
